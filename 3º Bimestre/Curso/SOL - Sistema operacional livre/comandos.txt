Find    : Procura por arquivos e diretórios diretamente no sistema de arquivos. Dependendo do tamanho do sistema de arquivos, esse comando pode ser muito demorado.
Locate  : Pode ser usado para encontrar arquivos no sistema através de uma base de dados que ele mantém, o resultado é instantâneo. Porém arquivos adicionados recentemente ao disco podem não estar na base de dados ainda.
Chown   : Este comando permite alterar o nome do dono e/ou do grupo de arquivos.
Chmod   : Recurso utilizado para trabalhar com permissões de arquivos do sistema Linux, cujos arquivos têm um sistema de sinalizadores. Esses são usados para indicar quais são as autorizações que cada usuário tem para editar ou ler determinado arquivo.
Netstate : Exibe as conexões de rede, tabela de rotas, estatísticas das interfaces etc. Ele foi elegantemente substituído pelo comando “ss“.
Traceroute : A função do programa é exibir a rota completa para mapear a transferência de um arquivo.
Reboot : Comando é usado para reiniciar ou reinicializar o sistema.
Shutdown : O comando shutdown pode reinicializar, parar ou desligar.
Power off : O comando shutdown é utilizado para desligar ou reiniciar o sistema com horários determinados.
