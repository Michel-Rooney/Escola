O ECG é uma placa econômica usada para medir a atividade elétrica do coração. Essa atividade elétrica 
pode ser mapeada como um ECG ou eletrocardiograma e gerada como uma leitura analógica. Os ECGs podem ser 
extremamente ruidosos, o Monitor de frequência cardíaca com eletrodo único AD8232 atua como um amplificador 
operacional para ajudar a obter um sinal claro dos intervalos PR e QT facilmente. Com ele pegamos os dados
da medição do coração 

ESP32 é uma série de microcontroladores de baixo custo e baixo consumo de energia. Também é um sistema-em-um-chip 
com microcontrolador integrado, Wi-Fi. Com os dados coletados do ECG, enviamos para o ESP, para comunicação com django

O framework Django, onde o mesmo é um framework web, open source baseando em python,
sua arquitetura é MVT, model (modelo de dados, ela faz a comunicação com o banco de dados), view (ela trabalho com
as requisições respodendo e retornando), template (onde fica a interface que será visivel ao usuario)
